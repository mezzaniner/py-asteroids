# Py-Asteroids

A basic version  of Asteroids - written in Python using Pygame lib.

I made this project in one day to refresh my python syntax knowledge. Enjoy!

## Usage
To run, use: 
```
uv run main.py
```

## Controls
`w` - move forward \
`s` - move backward \
`a` - turn left \
`d` - turn right \
`spacebar` - fire
